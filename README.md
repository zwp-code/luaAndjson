这是一个在线lua和json格式互转项目，支持lua table类型转换为json类型，支持json转换为lua table，支持文件导入导出，支持header/data模式和key/data模式的转换
This is an online Lua and JSON format conversion project that supports converting Lua table types to JSON types, converting JSON to Lua tables, importing and exporting files, and converting header/data and key/data modes
